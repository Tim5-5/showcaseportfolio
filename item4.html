GSC script

These are some the scripts I created for a passion project. The scripts are created with GSC which has some similarities with c++. GSC is used by the T7 engine and can be used to script anything. The first script is created so that the player can choose their own weapon camo in-game.

The game itself let’s player choose their weapon camo, but only in the menu. With this script the player can cycle between camo’s in-game, as long as the weapon is upgraded and a non-alt weapon.
